# Student Management System

A simple console-based Student Management System built with Python and Object-Oriented Programming. The project allows users to add, view, search, update, and delete student records from an interactive menu.

## Short Description

Python OOP-based Student Management System with CRUD operations, student search, marks update, and highest marks display.

## Features

- Add a new student record
- View all student records
- Search for a student using Student ID
- Update a student's marks
- Delete a student record
- Display the student with the highest marks
- Menu-driven console interface
- Uses classes and objects
- Stores multiple students using a list

## Student Details

Each student record contains:

- Student ID
- Student Name
- Age
- Marks

## Technologies Used

- Python
- Object-Oriented Programming
